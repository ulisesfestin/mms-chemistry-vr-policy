Política de Privacidad

Última actualización: 20 de noviembre de 2025

Esta Política de Privacidad está gestionada por la entidad desarrolladora del proyecto académico:
Universidad de Mendoza – Proyecto de Tesis MMS Chemistry VR.

MMS Chemistry VR es una aplicación educativa desarrollada como parte de una tesis universitaria. El proyecto no posee fines comerciales y no está diseñado para recopilar información personal de los usuarios.

1. Datos que recopila la aplicación

MMS Chemistry VR no recopila, almacena ni procesa datos personales de los usuarios. No se recopila ninguno de los siguientes tipos de datos:

  • Nombre, apellido o identificación personal
  • Correo electrónico o datos de contacto
  • Imágenes, grabaciones, voz o contenido generado por el usuario
  • Información biométrica
  • Datos de ubicación o GPS
  • Datos financieros
  • Datos de rendimiento personalizados

La única información utilizada es aquella provista por el sistema Meta Quest de forma automática para permitir el funcionamiento técnico básico del dispositivo (como tracking, orientación del casco o métricas internas del sistema). MMS Chemistry VR no accede, procesa ni almacena estos datos.

2. Cómo usamos la información

Como la aplicación no recopila datos personales o identificables, no existe procesamiento, análisis, distribución ni uso de información del usuario.

Los datos técnicos manejados por el sistema operativo del Meta Quest son gestionados únicamente por Meta de acuerdo con sus propias políticas de privacidad, y MMS Chemistry VR no tiene acceso ni control sobre ellos.

3. Solicitud de eliminación de datos

Dado que la aplicación no recopila ni almacena datos de los usuarios, no existe información que pueda ser eliminada o recuperada.

Si tenés alguna inquietud acerca de privacidad o deseas verificar que no se almacena información, podés contactarnos a:
ulises_festin@hotmail.com

4. Protección de datos

Aunque la aplicación no recopila datos personales, se aplican las siguientes medidas:

  • No se transmiten datos a servidores externos.
  • No se utiliza almacenamiento local para información del usuario.
  • No se emplean SDKs de terceros para analytics o publicidad.
  • No se comparte información con terceros bajo ninguna circunstancia.
